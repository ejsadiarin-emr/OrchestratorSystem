# Decision: Init Steps Frontend Scope

**Date:** 2026-04-30
**Status:** Resolved

## Q16: Frontend Support for Init Steps in PoC

**Decision:** Option A — Frontend reads init steps (display only).

- Workload detail page shows init steps for each package in read-only mode
- No UI to create/edit init steps in this phase
- Workload JSON import is the authoring path for init steps
- Full CRUD editing UI can come in a future phase
- This allows end-to-end data flow verification without significant frontend effort
